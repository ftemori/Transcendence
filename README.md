# Usage (for Dockers)

1. In the root directory, run docker compose up --build

# Usage for the frontend (for just the game)

1. go to ./frontend
2. run `pnpm install`
3. run `pnpm start` to compile typescript and serve the game with a simple http-server
4. run `pnpm run dev` to compile typescript with watch mode + live-server (self reloading)
